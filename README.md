# GITA (Gimana Tadi?)

GITA is a minimalist, mobile-first mood tracking and journaling application designed for simplicity and emotional clarity.

## Features

- **Mood Tracking**: Log your daily emotions with custom sticker-style icons.
- **Journaling**: Securely record your daily stories and thoughts.
- **Monthly Calendar**: Visualize your emotional trends over time with a comprehensive grid view.
- **Wawasan (Insights)**: Track your writing streaks and mood distribution.
- **Offline First**: All your data is stored locally using Hive for privacy and speed.

## Tech Stack

- **Framework**: Flutter
- **State Management**: Flutter Riverpod
- **Local Database**: Hive
- **UI Components**: Custom Design System, Table Calendar, Google Fonts

## Getting Started

1. Clone the repository.
2. Run `flutter pub get` to install dependencies.
3. Run `dart run build_runner build` to generate Hive adapters.
4. Run `flutter run` to launch the application.
